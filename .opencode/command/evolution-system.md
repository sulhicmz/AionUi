# Self-Evolution Commands

## /evolve-cycle

Trigger a full evolution cycle to analyze performance, apply optimizations, and update knowledge base.

## /heal-system

Run comprehensive diagnostics and apply self-healing protocols to detected issues.

## /learn-patterns

Analyze recent operations to identify patterns and update learning systems.

## /optimize-agents

Optimize agent selection and task distribution based on performance history.

## /memory-dump

Export current memory state, knowledge base, and evolution logs for analysis.

## /memory-restore

Import previously saved memory state to restore system configuration.

## /diagnostic-report

Generate a comprehensive diagnostic report of all system components.

## /benchmark-agents

Run performance benchmarks on all agents to identify optimization opportunities.

## /auto-tune

Enable automatic tuning of system parameters based on usage patterns.

## /reset-evolution

Reset evolution history while preserving core knowledge (use with caution).
